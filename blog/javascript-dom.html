JavaScript and the DOM

Let's talk javaScript, HTML and CSS

Javascript, HTML and CSS all work together to create a dynamic webpage.

HTML is like the skeleton of a human body. CSS is the skin and appearance of the human body. JavaScript is the nervous system of the body. It enables coordination between different parts of the body. It allows the human body to create actions as a response to external input.

Control flow and loops

Control flow is the order in which the javascript code is executed. Loops allow you to repeat a certain part of the code until a certain condition is met

I am going to use the activity of going to the gym as an example to explain control flow and loops.

The control flow:

  1. Drive to the gym
  2. Swipe entry card
  3. Fill up water bottle
  4. Warmup and stretch
  5. Perform the strength training workout
  6. Cool down and stretch
  7. Exit the gym
  8. Drive home
