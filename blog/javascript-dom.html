Javascript and the DOM

Let's talk javascript, HTML and CSS

Javascript, HTML and CSS are all important parts of creating a dynamic webpage. So what role does each of these languages play?

Lets imagine the human body. HTML is would be the skeleton of a human body, it provides the structure of a website. CSS would be the skin and appearance of the human body, it provides the styling of a website. Javascript would be the nervous system of the body. It enables coordination between different parts of the body and allows the human body to create actions as a response to different inputs. It adds functionality to a website.

Control flow and loops

Control flow and loops are fundamental concepts of javascript. Control flow is the order in which the javascript code is executed. Loops allow you to repeat a certain part of the code until a certain condition is met

I am going to use the example of going to the gym to explain control flow and loops.

The control flow:

  1. Drive to the gym
  2. Swipe entry card
  3. Fill up water bottle
  4. Warmup and stretch
  5. Perform the strength training workout
  6. Cool down and stretch
  7. Exit the gym
  8. Drive home

The DOM

The DOM (document object model) is another essential concept in web development. The DOM is a tree-like representation of a webpages HTML document. It allows Javascript to access and manipulate elements in a webpage.

An example of how you might interact with the DOM is using document.getElementById() which retrieves an element from the DOM using its unique ID.
